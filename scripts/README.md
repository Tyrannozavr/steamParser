# Скрипты и примеры

Эта директория содержит вспомогательные скрипты для отладки, тестирования и примеры использования.

## Python скрипты

- `analyze_html.py` - Анализ HTML страниц Steam Market
- `debug_api_requests.py` - Отладка API запросов
- `debug_parser.py` - Отладка парсера
- `demo_requirements.py` - Демонстрация требований
- `parse_item_example.py` - Пример парсинга предмета
- `show_urls.py` - Показ URL для отладки
- `debug_page.html` - HTML страница для отладки

## JavaScript скрипты (js/)

Скрипты для перехвата запросов расширения браузера:

- `intercept_background_fetch.js` - Перехват fetch запросов в background script
- `intercept_background_messages.js` - Перехват сообщений в background script
- `intercept_extension_requests.js` - Перехват запросов расширения
- `tracker.js` - Трекер запросов

## Использование

Эти скрипты используются для разработки и отладки. Они не являются частью основного приложения.
